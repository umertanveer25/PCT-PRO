# Contributing to PCT-PRO

First off, thank you for considering contributing to PCT-PRO! It's people like you that make the IoMT security community great.

## How Can I Contribute?

### Reporting Bugs
* Use the GitHub Issue Tracker.
* Provide a clear, descriptive title.
* Include steps to reproduce the issue.

### Suggesting Enhancements
* Open an issue to discuss your idea.
* Focus on clinical safety and protocol robustness.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the Google Style Guide.
3. Test your changes using the benchmarking suite in `scripts/`.
4. Update the documentation if you add new features.

## Community
Join our research discussions to help build a safer Internet of Medical Things.
