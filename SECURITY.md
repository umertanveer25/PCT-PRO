# Security Policy

## Supported Versions

We currently support the following versions of PCT-PRO with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this framework, please do not open a public issue. Instead, please report it directly to the authors:

*   **Yar Muhammad**
*   **Umer Tanveer**

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Scope
This policy applies to the core PCT-PRO engine and its associated protocol-gating mechanisms.
